## 과제 2: 파이썬 SW 개발

### 1. 프로그램 소개
- 제목: BMI 계산 및 건강 관리 프로그램
- 설명: 사용자에게 키와 현재 몸무게를 입력받아서 BMI 지수를 계산하고 상태를 알려줍니다. 그리고 목표 몸무게를 입력받아 몇 kg을 빼거나 늘려야 하는지 계산해 주고, 운동 루틴을 간단하게 출력해 줍니다. try-except 문법을 사용하여  예외 처리도 했습니다. 

### 2. 주요 기능
- 키와 몸무게를 활용한 BMI 계산 공식 적용
- if문을 사용해서 저체중, 정상, 과체중, 비만 단계 판정
- try-except를 사용해서 숫자가 아닌 문자가 입력되었을 때 에러가 나지 않도록 처리

### 3. 실행 방법
- python main.py

### 4. 자체 평가
- 수업 시간에 배운 조건문과 함수 기능을 연습하기 위해 작성했습니다. 입력값에 맞춰 몸무게 차이를 계산하고 간단한 운동 팁을 보여주도록 기능을 구성했습니다. 평소에 다이어트에 관심이 높고 어릴적에 비만으로 인하여 체중 20키로를 감량했던 적이 있어서 BMI 계산법에 관하여 찾아고 직접만들어 보고  git도 접해 볼 수 있어서 좋은 경험인 것 같습니다. 하지만 아직은 git에대해서 익숙하지 않아 더 많은 시간을 들여 사용해봐야 할 것 같습니다.
